# Usage

<!-- usage -->

```sh-session
$ npm install -g @modus/cli
$ modus COMMAND
running command...
$ modus (--version)
@modus/cli/0.0.0 linux-x64 node-v22.4.0
$ modus --help [COMMAND]
USAGE
  $ modus COMMAND
...
```
